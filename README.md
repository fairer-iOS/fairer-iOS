<h3 align="center">
 하우스워커가 선택한 첫번째 가사 관리 플랫폼
</h3>
<p align="center">
 여럿이서 함께 생활을 하다보면 집안일로 인해 갈등이 빚어지는 경우들이 생깁니다.<br>
 페어러는 이러한 집안일에 대한 어려움을 해소하고자 합니다.<br>
 집안일을 쉽게 관리하고 분담할 수 있도록 도와<br>
 모든 사람이 평화롭게 집안일을 하는 그 날까지 페어러가 함께합니다. 
</p>
